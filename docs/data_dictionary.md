## Data Dictionary (Core Fields)

This dictionary summarizes the key fields used throughout the report pipeline.

- `LMK_KEY`: Unique EPC certificate identifier  
- `LOCAL_AUTHORITY_LABEL`: Local authority name  
- `LODGEMENT_DATE`: Date EPC was lodged  
- `YEAR`: Lodgement year  
- `CURRENT_ENERGY_RATING`: EPC band (A–G)  
- `CURRENT_ENERGY_EFFICIENCY`: SAP score (1–100)  
- `POTENTIAL_ENERGY_EFFICIENCY`: Potential SAP score  
- `CO2_EMISSIONS_CURRENT`: Modelled CO2 emissions (tonnes/year)  
- `TOTAL_FLOOR_AREA`: Floor area (m²)  
- `PROPERTY_TYPE`: Dwelling type  
- `CONSTRUCTION_AGE_BAND`: Age band category  
- `MAIN_FUEL`: Raw main fuel label  
- `MAIN_FUEL_STD`: Standardised fuel class  
- `TENURE`: Raw tenure label  
- `TENURE_STD`: Standardised tenure class  
- `MAINS_GAS_FLAG`: On gas grid (Y/N)  
- `BELOW_C`: Boolean indicating EPC < C  
- `POLICY_PERIOD`: Policy cycle label (Pre-GreenDeal, ECO1, ECO2, MEES, Post-Strategy)
